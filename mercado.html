Los sistemas emergentes tienen numerosas aplicaciones, aquí exponemos algunas de las fuentes encontradas que utilizan los sistemas y tecnologias emergentes como metodo para resolucion de un problema o necesidad.

Tambien listaremos algunas posibles aplicaciones que como ingenieros se podrían modelar para dar solucion a un problema nacional.

Fuentes:

Medicina:

La medicina es un campo de acción bastante amplio,por lo cual es constantemente objeto de estudio y prueba para nuevos metodos de solución de problemas. Emerging Systems es una empresa dedicada a facilitar soluciones tecnologicas basadas en sistemas emergentes a entidades relacionadas con la medicina.

Previous Next

Aquí algunos casos de exito de Emerging Systems:

Cryosite requería un sistema de almacenamiento y rastreo basado en Internet para optimizar la administración y el monitoreo de componentes de datos extensos. Emerging Systems desarrolló un sistema llamado "Cryobyte" utilizando sistemas emergentes e inteligencia artificial.

South Australian Health Care buscaba de una Solución de Sistema de Información Clínica a nivel estatal, seleccionaron así el Sistema de Información Clínica EHS de Emerging Systems.

St. Vincent's & Mater Health requeria un sistema de información clínica habilitado para la web que debía proporcionar seguimiento del paciente desde el momento de la admisión hasta el alta.
